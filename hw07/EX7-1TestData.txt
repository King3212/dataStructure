下面是几个测试数据的例子：

1. 输入一个简单的3x3邻接矩阵：
3 3
0 1 2
1 0 3
2 3 0
-1 -1
 
期望输出：
卡鲁斯卡尔求最小生成树的结果序列
0 - 1 : 1
0 - 2 : 2
普里姆求最小生成树的结果序列
0 - 1 : 1
0 - 2 : 2
 

2. 输入一个4x4的邻接矩阵，包含负权边：
4 4
0 1 2 -1
-1 0 -3 4
2 3 0 5
1 -2 6 0
-1 -1
 

期望输出：
卡鲁斯卡尔求最小生成树的结果序列
1 - 2 : -3
0 - 1 : 2
1 - 3 : 4
普里姆求最小生成树的结果序列
0 - 1 : 2
1 - 2 : -3
1 - 3 : 4
 

3. 输入一个空矩阵：
0 0
-1 -1
 

期望输出：
卡鲁斯卡尔求最小生成树的结果序列

普里姆求最小生成树的结果序列
 

4. 输入一个只有一个节点的图：
1 1
0
-1 -1
 

期望输出：
卡鲁斯卡尔求最小生成树的结果序列
0
普里姆求最小生成树的结果序列
0
 

5. 输入一个对称矩阵：
3 3
0 1 2
1 0 3
2 3 0
-1 -1
 

期望输出：
卡鲁斯卡尔求最小生成树的结果序列
0 - 1 : 1
0 - 2 : 2
普里姆求最小生成树的结果序列
0 - 1 : 1
0 - 2 : 2
